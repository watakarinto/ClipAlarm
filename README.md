# Clip Alarm
好きな曲の好きな箇所を設定できるアラームアプリ。

## Dependency
* iOS14以上のiPhone
* 使用言語：Swift
* ライブラリ：[RangeSlider](https://github.com/hahnah/til-swift/tree/master/RangeSlider)

## Usage
プロジェクトの編集は、alarm.xcworkspaceから。</br>
アプリの初回立ち上げ時は、通知とミュージックのアクセスを許可する必要がある。
